# Coding Block Game - Flask + Unity WebGL

A computational thinking assessment game built with Unity WebGL and Flask backend.

## Features
- Unity WebGL game for computational thinking assessment
- Flask backend with OpenAI integration
- Student and teacher dashboards
- Real-time assessment and feedback
- PostgreSQL database

## Tech Stack
- **Frontend**: Unity WebGL
- **Backend**: Flask (Python)
- **Database**: PostgreSQL
- **AI**: OpenAI GPT-4

## Access Points
- Main Dashboard: `/`
- Unity Game: `/game`
- Teacher Panel: `/teacher/panel`
- Student Dashboard: `/student/dashboard`

## Environment Variables
- `OPENAI_API_KEY`: Your OpenAI API key
- `DATABASE_URL`: PostgreSQL connection string (auto-set by Railway)
